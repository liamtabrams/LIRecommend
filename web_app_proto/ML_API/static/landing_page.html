Welcome to LIRecommend!

This tool provides you with recommendations for job postings on LinkedIn after entering a search term, using a Random Forest Classifier model to predict what rating out of the possible options 0, 1, 2, or 3 the user would give to a given job posting. The dataset the model is trained on can be one you created from scratch or one of the pre-generated datasets that you can modify and build on. This tool also allows you to generate predictions for individual job postings by copying and pasting the correct posting URL into a box in the 'Recommend' page.
