Welcome to LIRecommend!

This tool provides rating predictions for job postings from LinkedIn. Simply enter the URL of a job posting, and the tool will predict the rating the user would give to the job. You can also collect data by submitting job URLs along with ratings to improve the prediction model. Please note that this tool only works when the job posting is opened in a new tab forcing the URL to start with "https://www.linkedin.com/jobs/view/" rather than "https://www.linkedin.com/jobs/collections/recommended/" which you get from viewing in LinkedIn's recommendation page.
